# reports

This directory is populated at runtime by the compatibility test runners.

| File                | Generated by             |
|---------------------|--------------------------|
| `test262.json`      | `runners/run-test262.ts` |
| `node-tests.json`   | `runners/run-node-tests.ts` |
| `web-tests.json`    | `runners/run-web-tests.ts` |
| `npm-tests.json`    | `runners/run-npm-tests.ts` |
| `summary.md`        | `runners/generate-report.ts` |

Run `npm run report` from `runtime/runners/` to regenerate `summary.md` from
the last set of saved JSON reports.

Run `npm run test:all` to execute every suite and then regenerate the summary.
