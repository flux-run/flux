# Wasm Runtime

> **Removed.** The Wasmtime-based Wasm execution path has been removed.
> Flux now supports only JavaScript/TypeScript via Deno V8.
>
> If you were using Wasm handlers, migrate to TypeScript. Use `flux serve <file>`
> to run any JS/TS file directly.

See [runtime.md](./runtime.md) for the current execution model.
