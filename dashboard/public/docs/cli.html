CLI Reference

Full reference for the flux command-line tool.

Installation

$ curl -fsSL https://fluxbase.co/install | bash
$ flux --version

flux login

Authenticate with Fluxbase. Opens a browser window and saves a token to ~/.config/fluxbase/config.toml.

$ flux login

flux create

Scaffold a new project from a built-in template.

$ flux create <name> [--template <template>]
FlagDescription
--template, -tTemplate slug. If omitted, shows interactive picker.

Available templates:

SlugDescription
todo-apiFull CRUD REST API with Postgres
webhook-workerWebhook receiver and event processor
ai-backendAI/LLM proxy with logging and caching
 
# Interactive picker
$ flux create my-app

# With template
$ flux create my-app --template todo-api

flux init

Link the current directory to an existing Fluxbase project.

$ flux init --project <project-id>

Creates a .fluxbase config file in the current directory.

flux deploy

Deploy functions to Fluxbase.

$ flux deploy [--name <fn-name>] [--runtime <runtime>]
  • Run from the project root to deploy all functions in functions/.
  • Run from inside a function directory (has flux.json) to deploy that single function.
FlagDescription
--nameOverride function name (single-function mode)
--runtimeOverride runtime (deno | node)

flux invoke

Invoke a deployed function.

$ flux invoke <name> [--payload <json>] [--gateway]
FlagDescription
--payloadJSON string to pass as input
--gatewayRoute through the API gateway (auth + rate-limiting)
 
$ flux invoke create_todo --payload '{"title":"Buy milk"}'

flux logs

Stream or tail function logs.

$ flux logs [--function <name>] [--tail] [--limit <n>] [--since <duration>]
$ flux logs --tail
$ flux logs --function create_todo --limit 50 --since 1h

flux trace

Render a distributed trace as a flame graph in your terminal.

$ flux trace <trace-id>

Trace IDs appear in log output and in the x-request-id response header. The trace shows:

  • Function execution span
  • Database query spans with duration
  • Cache hit/miss
  • N+1 pattern warnings
  • Slow query alerts (>100 ms)
  • Suggested indexes
$ flux trace c7f2a3e1-...

  Trace c7f2a3e1   2026-03-10 14:22:01 UTC

  ▸ function:create_todo           312 ms
    ▸ db:query(todos)               28 ms  [MISS]
    ▸ db:query(todos)               18 ms  [HIT]

  ─── Slow Queries ────────────────────────────────
    todos.user_id   284 ms   run: CREATE INDEX ON todos(user_id);

  ─── Index Suggestions ───────────────────────────
    → todos.user_id   run: CREATE INDEX ON todos(user_id);

flux secrets

Manage project secrets.

$ flux secrets set <KEY> <value>
$ flux secrets list
$ flux secrets delete <KEY>

Secrets are encrypted at rest and available in functions via ctx.secrets.get("KEY") or ctx.env.KEY.

flux project

Manage Fluxbase projects.

$ flux project create --name <name>
$ flux project list
$ flux project delete --id <id>

flux dev

Run a function locally with hot reload.

$ flux dev

flux doctor

Check CLI configuration and connectivity.

$ flux doctor

  ✔ Config file:        ~/.config/fluxbase/config.toml
  ✔ Authenticated:      you@example.com
  ✔ API reachable:      https://api.fluxbase.co  (42ms)
  ✔ Gateway reachable:  https://gw.fluxbase.co   (38ms)
