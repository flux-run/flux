From zero to deployed in one terminal session.

$ curl -fsSL https://fluxbase.co/install | bash
# → Fluxbase CLI 0.1.0 installed

$ flux login
# → Opens browser, saves token

$ flux create todo-api --template todo-api
# → Scaffolds functions/, schema/, README.md

$ cd todo-api && flux deploy
# → Deploys all 4 functions to Fluxbase
